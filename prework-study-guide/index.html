Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • lang= tells the browser the language for the webpage.
  • Style defines the CSS styles associated.
  • The base defines the base URL for the webpage.
  • The meta defines metadata such as character set, description, keywords, author, and viewpoints.
  • The script embeds client-side scripts in HTML documents.
  • The link connects an external resource to HTML document.
  • The charset sets our character encoding.
  • The http-equiv sets an HTTP header for content.
  • The viewpoint sets the default view for our webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations contain two important components:
  • The CSS property we want to apply.
  • The value of the property.
  • The img selector property assigns property value to any image element in the markup.
  • ul declares font to be a certain pixel (px) size.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
